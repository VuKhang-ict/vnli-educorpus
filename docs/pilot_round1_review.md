# Pilot Round 1 Review

## Mục tiêu
Kiểm tra workflow Label Studio pilot đầu tiên:
- project tạo được
- XML config hoạt động
- import JSON thành công
- annotate được
- export JSON thành công

## Kết quả đạt được
- 10 task đầu tiên đã được annotate đầy đủ
- Các trường bắt buộc lưu thành công: nli_label, confidence, rationale
- Export JSON thành công

## Vấn đề phát hiện
- Mẫu pilot round 1 chưa cân bằng nhãn
- Chưa kiểm tra được ranh giới giữa ENT, CON, NEU
- Chưa có double annotation
- Chưa có adjudication

## Kết luận
Pilot round 1 đạt mục tiêu kiểm tra hạ tầng và giao diện.
Cần thực hiện pilot round 2 với tập mẫu cân bằng nhãn và có đối sánh giữa hai lượt gán nhãn.
