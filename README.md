# VNLI-EduCorpus

VNLI-EduCorpus là dự án xây dựng bộ dữ liệu Suy luận ngôn ngữ tự nhiên cho tiếng Việt trong lĩnh vực giáo dục.

## Mục tiêu
- Xây dựng dataset premise-hypothesis-label cho miền giáo dục
- Ưu tiên Ngữ văn THPT theo định hướng GDPT 2018
- Hỗ trợ luận văn thạc sĩ và định hướng public dataset sau này

## Nhãn
- ENT = Entailment (kéo theo)
- CON = Contradiction (mâu thuẫn)
- NEU = Neutral (trung tính)

## Cấu trúc chính
- `data/metadata/`: file theo dõi nguồn và schema
- `data/samples/`: dữ liệu mẫu
- `notebooks/`: notebook Colab
- `docs/`: tài liệu mô tả quy trình
- `scripts/`: script xử lý sau này

## Ghi chú pháp lý
- Không sao chép nguyên văn sách giáo khoa hay tài liệu có bản quyền
- Chỉ dùng nguồn hợp pháp hoặc nội dung tự biên soạn hoặc diễn giải hợp lệ
